# インストール

## go install

```bash
go install github.com/fruitriin/ccchain/cmd/ccchain@latest
```

## ソースからビルド

```bash
git clone https://github.com/fruitriin/EnumaElish.git
cd EnumaElish
make build
# ./ccchain にバイナリが生成されます
```

## GitHub Releases

macOS / Linux 向けのビルド済みバイナリを [Releases ページ](https://github.com/fruitriin/EnumaElish/releases)で配布しています。

## 確認

```bash
ccchain --version
```
